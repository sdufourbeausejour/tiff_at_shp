# tiff_at_shp

tiff_at_shp is a Python script to sample a TIFF image
at shapefile feature points

## Usage

```python
import tiff_at_shp

tiff_at_shp.pixel_values(image_path, shapefile_path, results_dir, band_names) # get tiff values at all features in shp
```
